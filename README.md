# button-type-mu





```bash
npm install --save button-type-ui
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'button-type-ui'
import 'button-type-ui/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
